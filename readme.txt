Add User Role Label Next to Comments in WordPress

== Description ==

 Displaying user role label gives weight to comments made by registered users on your website specifically authors, editors, and admins. In this article, by using this plugin you will be able to easily add user role label next to comments in WordPress.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
